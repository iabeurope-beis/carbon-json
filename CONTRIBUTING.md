# Contributing

This repository is open for public feedback on the carbon.json public-feedback draft.

Feedback may be submitted through GitHub Issues or by email to IAB Europe's Data & Innovation Strategist, Dimitris Beis, at beis [at] iabeurope [dot] eu. The public feedback period is open until 19 August 2026.

Pull requests are welcome for clear editorial fixes, broken links, schema formatting issues, or example validation problems. Substantive methodology changes should normally be raised as Issues first.

All accepted changes are subject to review by the maintainers.
