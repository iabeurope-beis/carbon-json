# Public Feedback

Status: Public feedback draft

Version: v1.0 public-feedback draft

Intended use: Voluntary disclosure and implementation testing; not yet a final conformance standard.

## Feedback Period

Feedback deadline: 19 August 2026

Feedback contact: IAB Europe's Data & Innovation Strategist, Dimitris Beis, at beis [at] iabeurope [dot] eu.

## How to Comment

Please use GitHub Issues for public comments where possible:

- use the public feedback issue template for methodology or scope comments;
- use the schema issue template for validation or field-structure problems; and
- use the implementation question template for practical interpretation questions.

Feedback may also be sent by email to IAB Europe's Data & Innovation Strategist, Dimitris Beis, at beis [at] iabeurope [dot] eu.

## Useful Feedback

The most useful comments identify:

- the affected section or schema field;
- the issue or implementation concern;
- proposed replacement wording or schema structure;
- rationale for the change;
- expected impact on comparability, feasibility, data quality, or validation; and
- stakeholder context, where relevant.

## Pull Requests

Pull requests are welcome for editorial fixes, broken links, schema formatting issues, and example validation problems.

Substantive methodology changes should normally be raised as Issues first so that the rationale, implementation impact, and alternative approaches can be discussed before wording changes are proposed.

All accepted changes are subject to maintainer review.
